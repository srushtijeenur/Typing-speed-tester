# Typing Speed Tester

A simple web app to test your typing speed using HTML, CSS, and JavaScript.

# Features
- Random paragraph generation
- WPM calculation
- Timer display
- Stop & Reset functionality
- Final result popup

# Tech Stack
- HTML
- CSS
- JavaScript

# How to Run
1. Download the files
2. Open `index.html` in a browser
